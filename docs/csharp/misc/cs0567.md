# 编译器错误 CS0567
接口不能包含运算符  
  
 [接口](../../csharp/language-reference/keywords/interface.md)定义中不允许使用运算符。  
  
 下面的示例生成 CS0567：  
  
```  
// CS0567.cs interface IA { int operator +(int aa, int bb);   // CS0567 } class Sample { public static void Main() { } }  
```
