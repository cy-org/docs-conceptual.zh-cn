# 编译器错误 CS0576
命名空间“namespace”包含与别名“identifier”冲突的定义  
  
 尝试两次使用同一个命名空间。  
  
## 示例  
 以下示例生成 CS0576：  
  
```  
// CS0576.cs using SysIO = System.IO; public class SysIO { public void MyMethod() {} } public class Test { public static void Main() { SysIO.Stream s;   // CS0576 } }  
```
