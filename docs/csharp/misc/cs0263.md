# 编译器错误 CS0263
“type”的分部声明必须指定相同的基类  
  
 在分部声明中定义类型时，请在每个分部声明中指定相同的基类。 有关详细信息，请参阅[分部类和方法](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md)。  
  
 下面的示例生成 CS0263：  
  
```  
  
// CS0263.cs // compile with: /target:library class B1 { } class B2 { } partial class C : B1  // CS0263 – is the base class B1 or B2? { } partial class C : B2 { }  
```
