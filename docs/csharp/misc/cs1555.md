# 编译器错误 CS1555
无法找到为“Main”方法指定的“class”  
  
 已向 [\/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 编译器选项指定了一个类，但无法在源代码中找到该类的名称。
