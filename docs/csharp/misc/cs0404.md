# 编译器错误 CS0404
意外的“\<”: 特性不能是泛型的  
  
 特性中不允许使用泛型类型参数。 删除类型参数和尖括号。  
  
 下面的示例生成 CS0404：  
  
```  
// CS0404.cs [MyAttrib<int>]  // CS0404 class C { public static void Main() { } }  
```
