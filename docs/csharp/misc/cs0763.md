# 编译器错误 CS0763
两个分部方法声明必须都是静态声明，或者两者都不能是静态声明。  
  
 分部方法声明不能有一个部分为静态，而其他部分为非静态。  
  
### 更正此错误  
  
1.  两个部分必须同时为静态或非静态。  
  
## 示例  
 下面的代码生成 CS0763：  
  
```  
// cs0763.cs using System; public partial class C { static partial void Part(); partial void Part() // CS0763 { } public static int Main() { return 1; } }  
```
