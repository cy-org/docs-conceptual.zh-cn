# 编译器错误 CS1609
修饰符不能放置在事件访问器声明上  
  
 修饰符仅能放置在事件声明上，而不能放在事件访问器声明上。 有关更多信息，请参见[使用属性](../../csharp/programming-guide/classes-and-structs/using-properties.md)。  
  
## 示例  
 以下示例生成 CS1609。  
  
```  
// CS1609.cs // compile with: /target:library delegate int Del(); class A { public event Del MyEvent { private add {}   // CS1609 // try the following line instead // add {} remove {} } }  
```
