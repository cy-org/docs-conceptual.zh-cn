# 编译器错误 CS0543
“enumeration”：枚举器值太大，不能适应它的类型  
  
 已赋给[枚举](../../csharp/language-reference/keywords/enum.md)中某元素的值超出该数据类型。  
  
 以下示例生成 CS0543：  
  
```  
// CS0543.cs namespace x { enum I : byte {a = 255, b, c}   // CS0543 public class clx { public static int Main() { return 0; } } }  
```
