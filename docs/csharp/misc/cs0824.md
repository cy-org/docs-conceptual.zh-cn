# 编译器警告（等级 1）CS0824
构造函数“name”标记为外部对象。  
  
 构造函数可能被标记为外部对象。 但是，编译器无法验证构造函数是否实际存在。 因此会生成警告。  
  
### 删除此警告  
  
1.  使用杂注警告指令来忽略它。  
  
2.  将构造函数移到类型内。  
  
## 示例  
 以下代码生成 CS0824：  
  
```  
// cs0824.cs public class C { extern C(); // CS0824 public static int Main() { return 1; } }  
```
