# 编译器错误 CS1732
预期参数。  
  
 当 lambda 表达式包含一个跟在输入参数后的逗号而未指定以下参数时，则会产生此错误。  
  
### 更正此错误  
  
1.  请删除此逗号，或添加编译器预期逗号后应有的输入参数。  
  
## 示例  
 下面的示例生成 CS1732：  
  
```  
// cs1732.cs // compile with: /target:library class Test { delegate void D(int x, int y); static void Main() { D d = (x,) => { }; // CS1732 } }  
```
