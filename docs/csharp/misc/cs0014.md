# <a name="compiler-error-cs0014"></a>编译器错误 CS0014
找不到所需的文件 “file”  
  
 编译器所需的某个文件不在系统上。 请确保路径正确无误。 如果文件是[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]系统文件，则您可能需要进行修复安装或其他删除和重新安装[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]完全。
