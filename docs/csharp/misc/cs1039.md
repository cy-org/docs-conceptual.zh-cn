# 编译器错误 CS1039
字符串未终止  
  
 编译器检测到格式不正确的[字符串](../../csharp/language-reference/keywords/string.md)文本。  
  
## 示例  
 下面的示例生成 CS1039。 若要解决此错误，请添加终止引号。  
  
```  
// CS1039.cs public class MyClass { public static void Main() { string b = @"hello, world;   // CS1039 } }  
```
