# 编译器错误 CS2036
要使用 \/pdb 选项，必须同时也使用 \/debug 选项。  
  
 仅为调试版本生成程序数据库文件。 因此，**\/pdb** 选项在零售版本中没有意义。  
  
### 更正此错误  
  
-   添加 **\/debug** 编译器选项。  
  
-   删除 **\/pdb** 编译器选项。  
  
## 示例  
 当使用 **\/pdb** 选项但不使用 \/debug 选项对其进行编译时，下面的示例生成 CS2036：  
  
```  
// cs2036.cs // Compile with: /pdb // CS2036 class Test { public static int Main() { return 1; } }  
```
