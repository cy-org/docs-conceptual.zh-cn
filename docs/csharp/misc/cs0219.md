# 编译器警告（等级 3） CS0219
变量 “variable” 已赋值，但其值从未使用过  
  
 当你声明并给一个变量赋值却不使用该变量时，编译器会发出一个 3 级警告。  
  
 以下示例生成 CS0219：  
  
```  
// CS0219.cs // compile with: /W:3 public class MyClass { public static void Main() { int a = 0;   // CS0219 } }  
```
