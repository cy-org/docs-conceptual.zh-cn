# 编译器错误 CS1605
无法将“var”作为 ref 或 out 参数传递，因为它是只读的  
  
 一个作为 [ref](../../csharp/language-reference/keywords/ref.md) 或 [out](../../csharp/language-reference/keywords/out.md) 参数传递的变量应在调用的方法中修改。 因此，不可能将只读参数作为 `ref` 或 `out` 进行传递。
