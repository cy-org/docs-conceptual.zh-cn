# 编译器错误 CS1103
扩展方法的第一个参数的类型不能是“type”。  
  
 扩展方法的第一个参数的类型不能是指针类型。  
  
## 示例  
 下面的示例生成 CS1103：  
  
```  
// cs1103.cs public static class Extensions { public unsafe static char* Test(this char* charP) { return charP; } // CS1103 }   
```
