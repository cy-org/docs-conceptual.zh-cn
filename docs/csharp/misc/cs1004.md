# 编译器错误 CS1004
重复的“modifier”修饰符  
  
 检测到重复的修饰符，如**访问**修饰符。  
  
 下面的示例生成 CS1004：  
  
```  
// CS1004.cs public class clx { public public static void Main()   // CS1004, two public keywords { } }  
```
