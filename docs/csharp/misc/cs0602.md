# 编译器警告（等级 1）CS0602
功能“old\_feature”已弃用。 请改为使用“new\_feature”  
  
 虽然目前仍然支持 \(*old\_feature*\) 代码中使用的语言功能，但在以后的版本中可能会删除该支持。 应转而使用建议的语法 \(*new\_feature*\)。
