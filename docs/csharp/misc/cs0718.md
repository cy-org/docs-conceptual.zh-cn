# 编译器错误 CS0718
“type”：静态类型不能用作类型参数  
  
 因为静态类型不能被实例化，它不能用作泛型参数。 若要解决此错误，请从泛型参数中删除静态类型。  
  
## 示例  
 以下示例生成 CS0718：  
  
```  
// CS0718.cs public static class SC { public static void F() { } } public class G<T> { } public class CMain { public static void Main() { G<SC> gsc = new G<SC>();  // CS0718 } }  
```
