# <a name="endregion-c-reference"></a>#endregion（C# 参考）
`#endregion` 标记 [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) 块的结尾。 例如:   
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
