# <a name="help--c-compiler-options"></a>/help，/? （C# 编译器选项）
此选项将一列编译器选项和每个选项的简短说明发送到 StdOut。  
  
## <a name="syntax"></a>语法  
  
```console  
/help  
/?  
```  
  
## <a name="remarks"></a>备注  
 如果编译中包含此选项，则不会创建任何输出文件，且不会发生任何编译。  
  
 此编译器选项在 Visual Studio 中不可用，并且无法以编程方式更改。
