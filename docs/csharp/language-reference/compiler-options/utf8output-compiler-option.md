# <a name="utf8output-c-compiler-options"></a>/utf8output（C# 编译器选项）
/utf8output 选项使用 UTF-8 编码来显示编译器输出。  
  
## <a name="syntax"></a>语法  
  
```console  
/utf8output  
```  
  
## <a name="remarks"></a>备注  
 在某些国际配置中，编译器输出无法在控制台上正确显示。 在这些配置中，请使用 /utf8output 并将编译器输出重定向到文件。  
  
 此编译器选项在 Visual Studio 中不可用，并且无法以编程方式更改。
