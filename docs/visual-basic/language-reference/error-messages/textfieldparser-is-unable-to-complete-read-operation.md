# TextFieldParser 无法完成读取操作，因为已经超出最大缓冲区大小
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

该操作无法完成，原因是已超出最大缓冲区大小（10,000,000 字节）。  
  
### 更正此错误  
  
-   确保文件中没有格式错误的字段。
