# 常量表达式无法在类型“&lt;类型名&gt;”中表示
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

常数的计算结果与目标类型不符，通常是因为结果溢出范围。  
  
 **错误 ID：**BC30439  
  
### 更正此错误  
  
1.  更改目标类型，使其可以处理该常数。
