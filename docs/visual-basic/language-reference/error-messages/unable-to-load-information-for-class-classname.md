# <a name="unable-to-load-information-for-class-39ltclassnamegt39"></a>无法加载类的信息&lt;classname&gt;
一个引用了不可用的类。  
  
 **错误 ID:** BC30712  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证定义了类和名称拼写正确。  
  
2.  尝试访问该模块中声明的其中一个成员。 在某些情况下，调试环境找不到成员，因为尚未加载在其中声明成员的模块。
