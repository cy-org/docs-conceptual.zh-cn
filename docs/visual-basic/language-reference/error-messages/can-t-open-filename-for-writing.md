# 无法打开“&lt;文件名&gt;”进行写入
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

无法打开指定的文件进行写入，可能是因为它已经打开。  
  
 **错误 ID：**BC2012  
  
### 更正此错误  
  
1.  关闭文件然后重新打开它。  
  
2.  检查文件的权限。
