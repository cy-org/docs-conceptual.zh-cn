# 类型不匹配 (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

尝试使用无效方法将值转换为另一种类型。  
  
### 更正此错误  
  
1.  检查赋值语句确保其有效。  
  
2.  确保未将对象传递给要求单个属性或值的过程。  
  
3.  确保未在需要表达式的地方使用模块或项目名称。
