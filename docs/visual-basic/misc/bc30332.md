# 由于“&lt;type3&gt;”不是派生自“&lt;type4&gt;”，因此类型“&lt;type1&gt;”的值无法转换为“&lt;type2&gt;”。
当两个数组元素的数据类型并非同为引用类型时，或无法在两个数组的元素类型之间进行扩大转换或收缩转换时，语句试图将一个数组类型转换为另一个数组类型。  
  
 **错误 ID：**BC30332  
  
### 更正此错误  
  
-   检查两个数组的元素数据类型以确定错误源。
