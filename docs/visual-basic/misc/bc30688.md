# 接口中的事件不能声明为“&lt;implements&gt;”
在接口中声明的事件不能实现其他接口的事件。  
  
 **错误 ID：**BC30688  
  
### 更正此错误  
  
1.  删除 `Implements` 语句。  
  
2.  实现类或结构中的事件。
