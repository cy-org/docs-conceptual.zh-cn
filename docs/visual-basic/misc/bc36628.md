# 不能为变量“&lt;variablename&gt;”推断可以为 null 的类型
无法从引用类型（如数组、类或 `String` 推断可以为 Null 的类型。 用来推断数据类型的值必须具有值类型。 下面的代码演示此错误。  
  
```vb#  
'' Not valid.   
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **错误 ID：**BC36628  
  
### 更正此错误  
  
-   删除可以为 null 的指定。
