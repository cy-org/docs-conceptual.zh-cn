# “TypeOf ... Is”要求它的左操作数具有引用类型，但此操作数具有类型“&lt;type&gt;”
`TypeOf...Is` 表达式检查对象变量的运行时类型兼容性。 对于值类型未定义此兼容性。  
  
 **错误 ID：**BC30021  
  
### 更正此错误  
  
-   如果 `Option Strict` 为 `Off`，请使用 `TypeName` 或 `VarType` 函数来获取变量的数据类型信息。  
  
-   如果 `Option Strict` 为 `On`，则变量声明决定变量的数据类型。
