# 嵌入 Win32 清单时出错：选项 /win32manifest 与 /nowin32manifest 冲突
已向 Visual Basic 编译器传递 `/win32manifest` 编译器选项和 `/nowin32manifest` 编译器选项。 只有其中一个选项可以传递到 Visual Basic 编译器。  
  
 **错误 ID：**BC2033  
  
### 更正此错误  
  
1.  删除 `/win32manifest` 编译器选项或 `/nowin32manifest` 编译器选项。
