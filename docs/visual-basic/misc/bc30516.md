# 重载解决方法失败，因为没有可访问的“&lt;method&gt;”接受此数目的参数
你已调用重载方法，但编译器没有找到具有匹配参数列表的方法。  
  
 **错误 ID：**BC30516  
  
### 更正此错误  
  
-   检查所提供的参数列表。
