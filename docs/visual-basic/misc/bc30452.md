# 没有为类型“&lt;类型1&gt;”和“&lt;类型2&gt;”定义运算符“&lt;运算符名称&gt;”
你已尝试使用二元运算符执行操作，但该操作对于涉及两个返回值的代码元素无效。  
  
 **错误 ID：**BC30452  
  
### 更正此错误  
  
-   检查这两个元素，并确保它们互相兼容。
