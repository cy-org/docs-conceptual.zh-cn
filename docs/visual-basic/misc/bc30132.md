# 未定义标签“&lt;labelname&gt;”
引用了行标签或行号，但它没有在引用范围内定义。 标签必须在包含该引用的过程中。  
  
 **错误 ID：**BC30132  
  
### 更正此错误  
  
-   重构代码，使该行标签和引用在同一过程中。
