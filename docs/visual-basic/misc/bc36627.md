# lambda 表达式的计算在调试器中无效
在调试会话期间，已在“监视”或“即时”窗口中输入 lambda 表达式。 Lambda 表达式在调试窗口中不可用。  
  
 **错误 ID：**BC36627  
  
### 更正此错误  
  
-   在代码编辑器中测试 lambda 表达式
