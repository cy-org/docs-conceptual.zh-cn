# “&lt;method1&gt;”和“&lt;method2&gt;”无法相互重载，因为两者的差异仅在于声明为“ParamArray”的参数。
试图重载两种方法，它们的差异仅在于 `ParamArray` 参数的个数。 编译器将具有 `ParamArray` 参数的过程视为具有无限数目的重载，且各个重载的差异在于传递给参数数组的内容。  
  
 **错误 ID：**BC30368  
  
### 更正此错误  
  
-   请确保方法不仅由 `ParamArray` 参数进行区分。
