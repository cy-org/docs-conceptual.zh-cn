# “&lt;名称&gt;”不能公开事件的基础委托类型“&lt;委托类型&gt;”，它正通过&lt;标识符&gt;“&lt;类型&gt;”在项目外部实现
该代码在公共类外部公开了一个私有类型。  
  
 **错误 ID：**BC30915  
  
### 更正此错误  
  
-   将该类型声明为 `Public`。  
  
     \- 或 \-  
  
-   使用另一种类型。
