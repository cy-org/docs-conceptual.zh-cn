# “&lt;typename&gt;”不从 &quot;System.Attribute&quot; 继承，因此不能用作特性
试图使用并非从 `System.Attribute` 派生的类。  
  
 **错误 ID：**BC31504  
  
### 更正此错误  
  
1.  通过在类代码的第一行添加 `Imports` 语句，将自定义特性定义为从 `System.Attribute` 派生的类。
