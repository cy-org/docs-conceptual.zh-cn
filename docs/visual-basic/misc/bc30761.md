# 项目“&lt;projectname&gt;”不能引用项目“&lt;projectname&gt;”，因为“&lt;projectname&gt;”直接或间接引用“&lt;projectname&gt;”
项目包含对第二个项目的引用。 第二个项目又包含对引用它的项目的引用。  
  
 **错误 ID:** BC30761  
  
### 更正此错误  
  
-   删除其中一个引用。
