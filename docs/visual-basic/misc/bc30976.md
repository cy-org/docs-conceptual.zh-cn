# “Next”控制变量与“For”循环控制变量不匹配
`For...Next` 循环的 `Next` 语句中的控制变量必须与相应 `For` 语句中的变量匹配。  
  
 **错误 ID：**BC30976  
  
### 更正此错误  
  
-   检查 `Next` 语句和相应 `For` 语句中变量的拼写以确保其匹配。  
  
-   确保未在无意中删除此封闭循环的任何部分。  
  
-   如果此循环属于一组嵌套循环，请确保每个循环正常终止。
