# 接口中的委托不能声明为“&lt;specifier&gt;”
使用在接口中声明的委托指定了无效的修饰符。  
  
 **错误 ID：**BC31068  
  
### 更正此错误  
  
1.  删除修饰符，如 `Public`、`Private`、`Shared`、`Friend`、`Protected`、`Protected Friend` 和 `Overridable`。
