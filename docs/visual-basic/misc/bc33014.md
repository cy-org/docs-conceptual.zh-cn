# 运算符“&lt;operator&gt;”必须有一个参数
使用无参数或多个参数定义的一元运算符。  
  
 一元运算符必须只具有一个参数。  
  
 **错误 ID：**BC33014  
  
### 更正此错误  
  
-   调整定义以只指定一个参数。  
  
-   如果需要两个参数，则必须定义一个二元运算符。  
  
-   如果不需要参数或需要两个以上的参数，则必须使用 [Function 语句](../../visual-basic/language-reference/statements/function-statement.md) 来定义 `Function` 过程（而不是运算符）。
