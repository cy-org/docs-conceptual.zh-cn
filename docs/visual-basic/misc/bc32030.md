# “#ElseIf”不能作为“#If”块的一部分跟在“#Else”之后
`#ElseIf` 条件编译指令跟在 `#Else` 指令之后。`#Else` 必须为条件块中 `#End If` 指令之前的最后一个指令。  
  
 **错误 ID：**BC32030  
  
### 更正此错误  
  
1.  检查前面的 `#Else` 是否为 `#ElseIf`。  
  
2.  检查前面的 `#If` 块是否正确终止，以及新的 `#If` 块是否启动。  
  
3.  如果其他所有内容均正确无误，则将此 `#ElseIf` 指令及其相应的语句块移动到 `#Else` 块前面。
