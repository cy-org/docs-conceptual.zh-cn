# Try 必须至少有一个“Catch”或“Finally”
一个 `Try` 块必须包括一个 `Finally` 块或至少在 `End Try` 语句之前包括一个 `Catch` 块。  
  
 **错误 ID：**BC30030  
  
### 更正此错误  
  
-   在 `Try` 块中添加 `Catch` 或 `Finally` 块。
