# 模块“&lt;modulename&gt;”不能用作类型
你尝试将模块用作一种类型。 除非用作 `Get` 类型，否则不能使用模块。  
  
 **错误 ID：**BC30371  
  
### 更正此错误  
  
-   以可用作有效类型的内容替换该模块。
