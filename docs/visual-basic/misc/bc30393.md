# “Exit Try”只能出现在“Try”语句内
`Exit Try` 只能出现在 `Try` 块语句内。 有冗余的 `Exit Try` 语句，或者 `Exit Try` 语句出现在其对应 `Try` 块的边界之外。  
  
 **错误 ID：**BC30393  
  
### 更正此错误  
  
1.  找到并删除不必要的 `Exit Try` 语句。  
  
2.  将 `Exit Try` 语句移动到代码中适当的位置。
