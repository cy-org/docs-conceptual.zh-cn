# 应为逗号或“)”（过程参数列表）
参数列表包含意外的元素。 使用逗号 \(`,`\) 分隔此类列表中的参数，并使用右括号 \(`)`\) 结束列表。  
  
 **错误 ID：**BC30213  
  
### 更正此错误  
  
-   添加逗号继续类型形参列表，或添加右括号字符终止列表。
