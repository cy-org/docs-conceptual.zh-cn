# &quot;declare sub&quot; 语言元素中不允许出现 XML 注释标记 &quot;returns&quot;
declare sub 语言元素中不允许有 XML 注释标记 "returns"。 将忽略 XML 注释。  
  
 `Declare Sub` 声明的 XML 注释不能包含 `<`returns`>` 标记。  
  
 **错误 ID：**BC42315  
  
### 更正此错误  
  
-   删除不受支持的 `<`returns`>` 标记。
