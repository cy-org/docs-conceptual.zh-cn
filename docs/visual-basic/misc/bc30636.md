# 应为 &quot;&gt;&quot;
指定代码的语法不正确。 此错误通常表明某个特性或 XML 元素文本缺少右尖括号。  
  
 **错误 ID：**BC30636  
  
### 更正此错误  
  
-   检查所用代码的语法。
