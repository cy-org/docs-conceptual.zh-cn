# 参数“Acces”无效。 追加模式的有效值为“OpenAccess.Write”和“OpenAccess.Default”
访问值在追加模式中无效。  
  
### 更正此错误  
  
1.  将值更改为 `OpenAccess.Write` 或 `OpenAccess.Default`。
