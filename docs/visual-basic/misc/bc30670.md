# “ReDim”语句需要一个带括号的列表，该列表列出此数组每个维度的新界限
必须指定数组的新大小作为 `ReDim` 语句的一部分。  
  
 **错误 ID：**BC30670  
  
### 更正此错误  
  
-   提供该数组每级的大小；例如：  
  
    ```  
    ReDim arr(5, 6)  
    ```
