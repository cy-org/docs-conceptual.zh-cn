# 事件名称的长度不能超过 1011 个字符
使用 `Event` 语句声明的每个事件的名称或标识符限制为 1011 个字符。  
  
 **错误 ID：**BC32204  
  
### 更正此错误  
  
-   减小标识符的长度。
