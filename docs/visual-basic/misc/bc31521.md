# “&lt;attributename&gt;”不能多次应用于一个程序集
指定的特性只能对特性应用一次。  
  
 **错误 ID：**BC31521  
  
### 更正此错误  
  
1.  删除此特性的冗余应用。  
  
2.  如果使用你开发的自定义特性，请修改 `AttributeUsageAttribute` 并将 `AllowMultiple` 属性设置为 `True`。
