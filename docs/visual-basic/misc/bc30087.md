# “End If”前面必须是匹配的“If”。
出现 `End If` 语句而没有相应的 `If` 语句。`End If` 前面必须是 `If` 语句。  
  
 **错误 ID：**BC30087  
  
### 更正此错误  
  
1.  如果此 `If` 块属于一组嵌套的 `If` 块，请确保每个块均已正确终止。  
  
2.  验证 `If` 块中的其他控制结构是否被正确终止。  
  
3.  确保此 `If` 块的格式正确。
