# “Sub New”不能声明为“Overrides”
构造函数显示它重写继承构造函数。 不能重写构造函数。  
  
 **错误 ID：**BC30283  
  
### 更正此错误  
  
-   从 `Sub` 声明中删除 `Overrides` 关键字。
