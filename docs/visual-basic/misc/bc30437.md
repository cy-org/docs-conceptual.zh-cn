# “&lt;method1&gt;”无法重写“&lt;method2&gt;”，因为它们的返回类型不同
你已试图使用另一个返回类型不同的方法重写一个方法。 类型可重写继承的可重写方法，方法是声明具有相同名称和签名的方法，并用 `Overrides` 修饰符标记声明。 这两种方法的签名必须匹配。  
  
 **错误 ID：**BC30437  
  
### 更正此错误  
  
-   检查这两种方法的返回类型，并根据匹配需要进行更改。
