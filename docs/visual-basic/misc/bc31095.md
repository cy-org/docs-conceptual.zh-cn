# 调用另一个构造函数时引用尚未完成的对象是无效的
在对象的构造函数完成对象创建之前，已引用对象成员。  
  
 **错误 ID：**BC31095  
  
### 更正此错误  
  
-   从一个构造函数调用另一个构造函数时，请勿使用 `MyBase`、`MyClass` 或 `Me`。
