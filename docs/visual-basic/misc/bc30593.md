# 模块中的变量不能声明为“&lt;specifier&gt;”
你在 `Module` 语句中的某个变量上使用了一个说明符（如 `MustInherit`）。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID：**BC30593  
  
### 更正此错误  
  
-   删除说明符。
