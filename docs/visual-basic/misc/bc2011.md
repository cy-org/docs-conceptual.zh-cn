# 无法打开响应文件“&lt;filename&gt;”
你正试图访问的文件不可用。  
  
 **错误 ID：**BC2011  
  
### 更正此错误  
  
-   确保网络连接仍处于活动状态。  
  
-   检查你对该文件的权限。
