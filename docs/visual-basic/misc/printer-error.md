# 打印机错误
打印机发生错误，但未向发送文件的计算机返回其他信息。  
  
### 更正此错误  
  
-   对打印机进行物理检查。 确保计算机和打印机之间的所有连接牢固。 大多数打印机都会显示错误信息，例如“脱机”或“缺纸”。
