# 重载决策失败，因为没有可访问的“&lt;method&gt;”
调用了一个重载方法，但编译器找不到参数列表在参数个数上匹配的可访问方法。  
  
 **错误 ID：**BC30517  
  
### 更正此错误  
  
1.  检查所提供的参数列表。  
  
2.  检查试图调用的方法，以确定其是否可访问。
