# 无法将类型“&lt;type1&gt;”的参数“&lt;argumentname&gt;”转换到类型“&lt;type2&gt;”
转换操作尝试将参数转换为不兼容的类型。  
  
### 更正此错误  
  
1.  确保你尝试将参数转换到的类型有效。
