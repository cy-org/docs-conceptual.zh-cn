# “&lt;方法1&gt;”无法替代“&lt;方法2&gt;”，因为它扩展了基方法的访问权限
过程指定了 `Overrides`，但声明可访问性受到的限制少于将被重写的方法。 不能扩展可访问性，即你不能使重写的方法比被重写的方法更易于访问。 例如，如果基类方法是 `Protected`，则不能用 `Public` 方法重写此方法。  
  
 **错误 ID：**BC32203  
  
### 更正此错误  
  
-   删除 `Overrides` 关键字或更改可访问性，使其至少与基类方法具有同等限制。
