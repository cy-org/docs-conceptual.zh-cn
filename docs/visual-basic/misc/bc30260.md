# “&lt;name&gt;”已声明为此 &lt;declarationspace&gt; 中的“&lt;declaration&gt;”
在同一声明空间（类、模块、接口或结构）中，声明了两个同名编程元素。  
  
 **错误 ID：**BC30260  
  
### 更正此错误  
  
-   将其中一个声明移动到其他声明空间，或更改其中某个声明元素的名称。
