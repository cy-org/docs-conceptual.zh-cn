# &quot;End Set&quot; 前面必须是匹配的 &quot;Set&quot;
`End Set` 用于终止 `Set` 属性过程。`End Set` 构造出现在 `Set` 属性过程外。  
  
 **错误 ID：**BC30632  
  
### 更正此错误  
  
1.  请确保在 `Property` 关键字之后，`End Property` 构造之前声明 `Set` 属性过程。  
  
2.  确保 `Set` 属性过程以 `Set` 关键字开始，以 `End Set` 构造结束。
