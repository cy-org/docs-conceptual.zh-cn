# 语句不能出现在接口体内(Visual Basic 错误)
遇到意外的语言构造。 假定缺少 `End Interface` 构造，而且此点后的所有源代码都在接口外。  
  
 **错误 ID：**BC30604  
  
### 更正此错误  
  
1.  验证接口定义中所用代码的语法是否正确。  
  
2.  确保接口定义以 `End Interface` 构造结束。
