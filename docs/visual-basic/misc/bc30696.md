# “&lt;membername1&gt;”和“&lt;membername2&gt;”的差异仅在于可选参数的类型，因此它们无法重载对方
重载的成员必须为一个或多个所需参数提供不同数据类型。  
  
 **错误 ID：**BC30696  
  
### 更正此错误  
  
-   将至少一个所需参数定义为不同的数据类型。
