# 项目级条件编译常量“&lt;constantname&gt;”无效: &lt;error&gt;
为项目指定了无效的条件编译选项。  
  
 **错误 ID：**BC31030  
  
### 更正此错误  
  
1.  验证与此项目结合使用的条件编译常量。
