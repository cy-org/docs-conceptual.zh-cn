# 已用名称“&lt;argumentname&gt;”声明了参数
过程声明定义了具有相同名称的两个参数。 过程声明中的所有参数名称必须都是唯一的。  
  
 **错误 ID：**BC30237  
  
### 更正此错误  
  
-   更改其中一个参数的名称。
