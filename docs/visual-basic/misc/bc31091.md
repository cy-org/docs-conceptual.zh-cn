# 从程序集或模块“&lt;name&gt;”导入类型“&lt;typename&gt;”失败
请确保所有引用库彼此兼容。  
  
 **错误 ID：**BC31091  
  
### 更正此错误  
  
-   验证包含引用类型的文件未损坏。
