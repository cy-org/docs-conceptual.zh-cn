# XML 文档分析错误：开始标记“&lt;tag&gt;”没有匹配的结束标记
XML 文档分析错误：开始标记 \<tag\> 没有匹配的结束标记。 将忽略 XML 注释。  
  
 XML 注释包含开始标记，但不包含匹配的结束标记。  
  
 **错误 ID：**BC42316  
  
### 更正此错误  
  
-   添加与开始标记相匹配的结束标记。  
  
     — 或 —  
  
-   如果该标记不包含内部文本（如 [\<seealso\>](../../visual-basic/language-reference/xmldoc/seealso.md)），则在右尖括号前面指定一个正斜杠。
