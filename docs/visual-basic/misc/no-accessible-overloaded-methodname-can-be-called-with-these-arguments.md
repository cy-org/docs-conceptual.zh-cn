# 必须进行收缩转换才能用这些参数调用可访问的重载“&lt;methodname&gt;”
调用了重载方法，但是方法不可与所提供的未进行收缩转换的参数列表相匹配。  
  
### 更正此错误  
  
1.  指定 `Option Strict Off`。  
  
2.  更改参数以匹配重载方法的其中一个签名。
