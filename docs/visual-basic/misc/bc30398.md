# “&lt;method1&gt;”不能重写“&lt;method2&gt;”，因为它们的一个参数具有不同的标记，分别标记为“ByRef”和“ByVal”
已尝试使用具有不同参数的另一个方法重写了一个方法，该参数标记为 `ByRef` 而非 `ByVal`。 重写的成员必须具有与继承自基类的成员相同的参数。  
  
 **错误 ID：**BC30398  
  
### 更正此错误  
  
-   请确保参数或者都是 `ByRef`，或者都是 `ByVal`。
