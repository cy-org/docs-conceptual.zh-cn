# “&lt;name&gt;”不能通过 &lt;specifier2&gt;“&lt;type2&gt;”向&lt;specifier1&gt;“&lt;type1&gt;”公开它正在实现的事件的基础委托类型“&lt;delegatetype&gt;”中所用的类型“&lt;type&gt;”
该代码在公共类外部公开了一个私有类型。  
  
 **错误 ID：**BC30923  
  
### 更正此错误  
  
-   声明类型 `Public` 或使用另一种类型。
