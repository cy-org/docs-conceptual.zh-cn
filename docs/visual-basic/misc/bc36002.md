# <a name="39global39-not-allowed-in-handles-local-name-expected"></a>' Global' 句柄; 中不允许使用预期的本地名称
`Handles` 子句必须引用本地事件。 `Global` 关键字提供了对全局编程元素的访问权限。  
  
 **错误 ID：** BC36002  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Handles` 子句更改为引用事件的本地实例而不是全局实例。
