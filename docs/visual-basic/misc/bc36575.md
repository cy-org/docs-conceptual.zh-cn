# 匿名类型成员名前面必须有一个句点
在匿名类型声明的对象初始值设定项列表中，向其分配值的新成员名称前面必须有一个句点。 下面的示例演示了一个有效的和一个无效的声明：  
  
```vb#  
' Valid.  
Dim instanceName1 = New With {.memberName = 10}  
' Invalid declaration that causes this error.  
' Dim instanceName2 = New With {memberName = 10}  
```  
  
 **错误 ID：**BC36575  
  
### 更正此错误  
  
-   在成员名称前面添加一个句点。
