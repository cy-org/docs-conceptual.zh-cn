# 结构“&lt;structurename&gt;”不能包含自身的实例：&lt;error&gt;
结构声明一个变量并使用自身实例对其初始化。  
  
 结构可以包含其他结构的实例，但不能包含自身的内部实例。 尝试这样操作会导致无限递归。  
  
 **错误 ID：**BC30294  
  
### 更正此错误  
  
1.  检查声明语句中初始化表达式的拼写。  
  
2.  如果想要创建另一具有相同结构的实例，必须在结构之外声明和创建它。
