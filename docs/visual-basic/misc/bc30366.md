# “&lt;methodname&gt;”和“&lt;methodname&gt;”无法重载对方，应为它们的差异仅在于“ReadOnly”或“WriteOnly”。
试图重载两种方法，它们的差异仅在于 `ReadOnly` 和 `WriteOnly` 声明。 只能使用参数列表内的所有内容来区分版本。  
  
 **错误 ID：**BC30366  
  
### 更正此错误  
  
-   请确保方法由多个 `ReadOnly` 和 `WriteOnly` 区分。
