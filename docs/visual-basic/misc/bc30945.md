# 类型“&lt;typename&gt;”不能实现接口“&lt;interfacename&gt;”，因为它声明了具有返回类型的“&lt;eventsignature&gt;”
类或结构尝试实现某个接口，此接口声明了将返回值的事件。  
  
 Visual Basic 当前不支持声明将返回值的事件。  
  
 **错误 ID：**BC30945  
  
### 更正此错误  
  
-   从类或结构定义中删除 `Implements` 语句，或实现其他接口。
