# &quot;System.Runtime.InteropServices.DllImportAttribute&quot; 不能应用于 &quot;Get&quot; 或 &quot;Set&quot;
向 `Get` 或 `Set` 属性过程应用了 `DllImportAttribute` 特性。  
  
 **错误 ID：**BC31524  
  
### 更正此错误  
  
1.  将 `DllImportAttribute` 从 `Get` 和 `Set` 属性过程中删除。
