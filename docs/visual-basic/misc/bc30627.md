# “Option”语句必须位于所有声明或“Imports”语句之前
`Option` 语句应为你的源代码中第一个非注释语句。  
  
 **错误 ID：**BC30627  
  
### 更正此错误  
  
-   将 `Option` 语句移动到源代码顶部 `Imports` 和 `Namespace` 语句之前。
