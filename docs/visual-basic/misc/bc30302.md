# 在具有显式类型的声明中不能使用类型字符“&lt;character&gt;”
你在显式指定类型时使用了类型字符，如 `$`。  
  
 **错误 ID：**BC30302  
  
### 更正此错误  
  
-   删除此类型字符或从声明中删除 `As <Type>` 子句。
