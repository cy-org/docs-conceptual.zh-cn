# &quot;Finally&quot; 必须以匹配的 &quot;End Try&quot; 结束
代码中出现的 `Finally` 语句没有匹配的 `End Try`语句。`Finally` 语句必须以 `End Try` 语句结束。  
  
 **错误 ID：**BC30442  
  
### 更正此错误  
  
1.  删除 `Finally` 语句。  
  
2.  添加 `End Try` 语句来结束块。
