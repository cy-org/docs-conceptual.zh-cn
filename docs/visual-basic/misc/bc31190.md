# XML 文本和 XML 轴属性不可用
XML 文本和 XML 轴属性不可用。 添加对 System.Xml、System.Xml.Linq 和 System.Core 引用。  
  
 所编译的代码包含 XML 文本或 XML 轴属性，但是未引用编译 XML 文本或 XML 轴属性所需的程序集。  
  
 **错误 ID：**BC31190  
  
### 更正此错误  
  
-   添加对 System.Xml.dll、System.Xml.Linq.dll 和 System.Core.dll 程序集的引用。
