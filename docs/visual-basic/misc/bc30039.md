# 循环控制变量不能是属性或后期绑定索引数组
用于循环访问 `For` 循环的变量必须为数值数据类型。  
  
 **错误 ID：**BC30039  
  
### 更正此错误  
  
-   更改指定 `Integer`、`Short`、`Long`、`Byte`、`Single`、`Double` 或 `Decimal` 的循环控制变量的声明。
