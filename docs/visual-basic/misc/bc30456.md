# “&lt;name&gt;”不是“&lt;classname&gt;”的成员
已提供的成员不是类的成员。  
  
 **错误 ID:** BC30456  
  
### 更正此错误  
  
1.  检查成员名称以确保其准确无误。  
  
2.  使用类的实际成员。
