# “&lt;method1&gt;”和“&lt;method2&gt;”无法相互重载，因为两者差异仅在于参数声明为“ByRef”还是“ByVal”
已尝试用与前者区别仅在于声明为 `ByRef` 或 `ByVal` 的另一种方法重载某种方法。  
  
 **错误 ID：**BC30345  
  
### 更正此错误  
  
-   确保这两种方法的区别不仅在于 `ByRef` 或 `ByVal` 参数的名称。
