# “&lt;typename&gt;”包含声明为“MustOverride”的方法，因此它必须声明为“MustInherit”
包含 `MustOverride` 成员的类型必须声明为 `MustInherit`。  
  
 **错误 ID：**BC31411  
  
### 更正此错误  
  
-   将 `MustInherit` 修饰符添加到类型。
