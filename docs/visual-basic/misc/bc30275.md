# “&lt;specifier&gt;”在接口事件声明中无效
接口内的 `Event` 语句包含不允许的关键字，如 `Implements`。 接口只能定义成员，而不能实现它们。  
  
 **错误 ID：**BC30275  
  
### 更正此错误  
  
1.  从声明语句中删除关键字。  
  
2.  将接口成员的实现移动到实现该接口的类。
