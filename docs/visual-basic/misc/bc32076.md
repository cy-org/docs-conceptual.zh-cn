# “&lt;typename&gt;”是泛型类型，需要类型参数
变量、过程参数或函数返回声明具有泛型类类型或结构类型，但是此声明未提供任何类型参数。  
  
 按其性质，使用至少一个类型参数定义每个泛型类和结构。 当使用泛型类型来声明构造的类或结构时，必须为每个由泛型类型定义的类型形参提供一个类型实参。  
  
 **错误 ID：**BC32076  
  
### 更正此错误  
  
-   将类型列表添加到声明，用括号括起来并以 `Of` 关键字开头。
