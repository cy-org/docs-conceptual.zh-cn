# 远程进程的类型库或对象库的连接已丢失
你丢失了到远程进程的对象库或类型库的连接。  
  
### 更正此错误  
  
1.  重新启动 `Application` 对象。  
  
2.  在借其进入此帮助主题的错误对话框中，单击“确定”以显示“引用”对话框。 丢失的引用出现时左侧有“MISSING”字样。  
  
3.  删除丢失的引用。  
  
4.  在“引用”对话框中，选中 `Application` 对象的复选框。
