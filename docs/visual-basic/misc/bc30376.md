# “New”不能在类“&lt;classname&gt;”上使用，因为它包含尚未被重写的“MustOverride”成员
已在包含未被重写的 `MustOverride` 成员的类上尝试使用 `New`。  
  
 **错误 ID：**BC30376  
  
### 更正此错误  
  
-   删除 `New` 语句。
