# 应为“Is”。
在没有 `Is` 子句的情况下出现了 `TypeOf` 运算符。  
  
 **错误 ID：**BC30224  
  
### 更正此错误  
  
-   在 `TypeOf` 运算符后面添加 `Is` 子句；例如 `TypeOf MyControl Is ComboBox`。
