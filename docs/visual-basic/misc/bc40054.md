# 设计器生成的类型“&lt;type&gt;”中的“&lt;constructor&gt;”应调用 InitializeComponent 方法
设计器生成的类型中的构造函数不调用类型的 `InitializeComponent` 方法。  
  
 设计器生成的类型中的每个构造函数应调用类型的 `InitializeComponent` 方法。  
  
 **错误 ID：**BC40054  
  
### 更正此错误  
  
-   将调用添加到构造函数中的 `InitializeComponent` 方法。
