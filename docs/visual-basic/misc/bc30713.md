# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>退出语句中不能使用即时窗口
`Exit`语句仅可以在源代码中使用。  
  
 **错误 ID：** BC30713  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Exit` 语句。
