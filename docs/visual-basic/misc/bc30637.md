# Assembly 或 Module 特性语句必须位于文件中的任何声明之前
必须在源文件顶端，`Option` 和 `Imports` 语句之后、其他语句之前声明全局特性。  
  
 **错误 ID：**BC30637  
  
### 更正此错误  
  
1.  将全局特性（如 `<Module:>` 和 `<Assembly:>`）放在源文件的顶端。
