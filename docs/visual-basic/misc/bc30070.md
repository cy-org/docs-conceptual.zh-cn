# Next 控制变量与 For 循环控制变量“&lt;variablename&gt;”不匹配
`For...Next` 循环的 `Next` 语句中的控制变量必须与相应 `For` 语句中的变量匹配。  
  
 **错误 ID：**BC30070  
  
### 更正此错误  
  
1.  检查 `Next` 语句和相应 `For` 语句中变量的拼写以确保其匹配。  
  
2.  确保未无意中删除该封闭循环的任何部分。  
  
3.  如果此循环是一组嵌套循环的一部分，请检查每个循环是否正常终止。
