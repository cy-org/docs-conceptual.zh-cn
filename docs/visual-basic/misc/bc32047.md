# <a name="type-parameter-39lttypeparameternamegt39-can-only-have-one-constraint-that-is-a-class"></a>类型参数&lt;typeparametername&gt;只能有一个属于一个类约束
约束列表包括多个类。  
  
 类型形参上的约束列表可接受任意数量的接口，但最多能接受一个类。 提供有关该类型参数必须继承自该类的类型实参和[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]不支持多重继承。  
  
 **错误 ID：** BC32047  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   选择一个类，并使约束列表中只包括此类。  
  
-   你可定义其他类型形参，以容纳不可包含在此约束列表中的一个或多个类。
