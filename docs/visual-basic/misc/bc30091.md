# “Loop”前面必须是匹配的“Do”
出现 `Loop` 语句而没有相应的 `Do` 语句。`Loop` 前面必须有相应的 `Do` 语句。  
  
 **错误 ID:** BC30091  
  
### 更正此错误  
  
1.  如果此 `Do` 循环是一组嵌套 `Do` 循环的一部分，请确保每个循环正常终止。  
  
2.  验证 `Do` 循环内的其他控制结构是否被正确终止。  
  
3.  确保此 `Do` 循环的格式正确。
