# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>字段&lt;fieldname&gt;' 属于不受支持的类型
某个字段被声明为的类型，不能通过表示[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]。  
  
 **错误 ID：** BC30656  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将字段定义为支持的类型。  
  
2.  开发一种可以描述字段中数据的新类型。
