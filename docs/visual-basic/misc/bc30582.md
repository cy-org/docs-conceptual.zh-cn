# “SyncLock”操作数不能为“&lt;typename&gt;”类型，因为“&lt;typename&gt;”不是引用类型
通过确保多个线程不会同时执行同一条语句，`SyncLock` 语句允许在单个表达式上同步语句。`SyncLock` 语句中的表达式的类型必须是引用类型，如类、模块、接口、数组或委托。  
  
 **错误 ID：**BC30582  
  
### 更正此错误  
  
-   将类型更改为适当的引用类型。
