# “&lt;membername1&gt;”无法重写“&lt;membername2&gt;”，因为它们在可选参数类型上存在差异
重载的成员必须为一个或多个所需参数提供不同数据类型。  
  
 **错误 ID：**BC30697  
  
### 更正此错误  
  
-   将至少一个所需参数定义为不同的数据类型。
