# 特性成员“&lt;membername&gt;”不能作为赋值的目标，因为它未声明为“Public”
尝试给特性中的私有成员赋值。  
  
 **错误 ID：**BC31511  
  
### 更正此错误  
  
1.  删除赋值。  
  
2.  如果使用自己开发的自定义特性，将特性成员的访问修饰符更改为 `Public`。
