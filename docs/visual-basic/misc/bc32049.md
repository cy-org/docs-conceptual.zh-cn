# 类型参数已使用名称“&lt;类型参数名称&gt;”声明
泛型类型的类型参数名称使用与同一泛型类型的另一个类型参数相同的名称声明。  
  
 在泛型类型的类型形参列表中，每个类型形参的名称必须与以下所有名称均不同：  
  
-   相同类型形参列表中的每个其他类型形参，  
  
-   在任何包含泛型类型的类型形参列表中的每个类型形参，以及  
  
-   泛型类型本身的名称。  
  
 **错误 ID：**BC32049  
  
### 更正此错误  
  
-   重命名类型形参，使其有别于此帮助页上的列表中引用的每个名称。
