# 已多次包含响应文件 &lt;filename&gt;
指定了多次响应文件。 响应文件包含用于编译的编译器选项和源代码文件。  
  
 **错误 ID:** BC2003  
  
### 更正此错误  
  
-   删除多余的响应文件说明符。
