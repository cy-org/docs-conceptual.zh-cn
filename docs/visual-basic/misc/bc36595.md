# 方法不能同时包含“On Error GoTo”语句和 lambda 或查询表达式
某方法同时包含 `On Error Goto` 语句和 lambda 表达式或 LINQ 查询。 不能在一个方法中既包含 `On Error Goto` 语句又包含 lambda 表达式或 LINQ 查询。  
  
 **错误 ID：**BC36595  
  
### 更正此错误  
  
1.  用 `Try...Catch` 语句替换使用 `On Error Goto` 语句的异常处理代码。
