# 特性 &quot;Conditional&quot; 只在 &quot;Sub&quot; 声明中有效
<xref:System.Diagnostics.ConditionalAttribute> 特性已经应用于方法声明，而不是 `Sub` 过程。  
  
 **错误 ID：**BC41007  
  
### 更正此错误  
  
1.  将 <xref:System.Diagnostics.ConditionalAttribute> 从无效声明中删除。
