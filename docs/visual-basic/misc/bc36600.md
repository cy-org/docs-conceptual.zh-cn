# 已声明范围变量 &lt;变量&gt;
在 `Select` 子句，或 `Aggregate`、`Group By` 或 `Group Join` 子句的 `Into` 部分中指定的范围变量名称与已在查询子句中指定的范围变量名称重复。  
  
 **错误 ID：**BC36600  
  
### 更正此错误  
  
1.  确保查询子句中的所有范围变量都具有唯一的名称。
