# “#If” 块必须以匹配的 “#End If” 结束
`#If` 是条件编译指令。`#End If` 指令未终止 `#If` 块。  
  
 **错误 ID：**BC30012  
  
### 更正此错误  
  
-   添加 `#End If` 指令至条件编译块的末尾。
