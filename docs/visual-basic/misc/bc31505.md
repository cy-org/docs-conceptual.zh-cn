# “&lt;typename&gt;”没有“System.AttributeUsageAttribute”特性，因此不能用作特性
尝试使用一个在没有 `System.AttributeUsageAttribute` 的情况下声明的特性来定义其用法。  
  
 **错误 ID：**BC31505  
  
### 更正此错误  
  
1.  自定义特性必须是从应用了 `AttributeUsageAttribute` 特性的 `System.Attribute` 派生的类。
