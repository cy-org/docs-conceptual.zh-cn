# 类型“&lt;type1&gt;”的值无法转换为“&lt;type2&gt;”，因为“&lt;type3&gt;”不是引用类型
语句试图将一个数组类型转换为另一个数组类型，但目标数组元素的数据类型不是引用类型。  
  
 **错误 ID：**BC30333  
  
### 更正此错误  
  
-   检查两个数组的元素数据类型以确定错误源。
