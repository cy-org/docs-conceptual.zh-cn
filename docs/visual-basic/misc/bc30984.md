# 应为“=”(对象初始值设定项)
要在对象初始值设定项声明中为字段或属性建立初始值，必须使用等号。 例如，对于 `client` 的 `Name` 属性，以下声明将“Microsoft”作为初始值分配。  
  
```  
Dim client As New Customer { .Name = "Microsoft" }  
```  
  
 **错误 ID：**BC30984  
  
### 更正此错误  
  
-   在上面示例中显示的位置处添加等号。
