# &lt;error&gt;：“&lt;constructorname1&gt;”调用“&lt;constructorname2&gt;”
发生循环构造函数调用。 构造函数将调用 `Me.New()` 或 `MyClass.New()`。 一个可能的原因是尝试调用具有不同参数列表的重载构造函数。  
  
 **错误 ID：**BC30297  
  
### 更正此错误  
  
-   使用不同的参数列表调用重载构造函数。  
  
-   如果没有可访问的重载，则删除对 `Me.New()` 或 `MyClass.New()` 的调用。
