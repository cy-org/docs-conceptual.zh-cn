# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>错误语句中不能使用即时窗口
`Error`语句仅可以在源代码中使用。  
  
 **错误 ID：** BC30724  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从调试代码中删除 `Error` 语句。
