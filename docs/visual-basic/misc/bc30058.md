# 位于“Select Case”与第一个“Case”之间的语句和标签无效
非注释语句出现在开始的 `Select` 或 `Select Case` 语句和其第一个 `Case` 语句之间。  
  
 **错误 ID：**BC30058  
  
### 更正此错误  
  
-   如果插入语句是一条注释，则在它前面加上一个注释分隔符（`'` 或 `REM`）。 否则，移动或删除该语句。
