# 不允许通过用单个类型说明符声明多个变量来进行显式初始化
当在同一行声明多个变量时，不允许进行初始化。  
  
 **错误 ID：**BC30671  
  
### 更正此错误  
  
1.  分别声明并初始化每个项。  
  
2.  一起声明多个项，然后初始化每个项；例如：  
  
    ```  
    Dim x, b, i As Integer x = 9 : b = 9 : i = 9 ' ":" is the same as a new line.  
    ```
