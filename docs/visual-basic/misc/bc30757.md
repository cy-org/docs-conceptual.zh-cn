# “GoTo &lt;labelname&gt;”无效，因为“&lt;labelname&gt;”位于不包含此语句的“For”或“For Each”语句内
`GoTo` 语句限制为在当前代码块中跳转。  
  
 **错误 ID：**BC30757  
  
### 更正此错误  
  
-   重构你的代码，以便 `GoTo` 语句和标签都位于 `For` 块中。
