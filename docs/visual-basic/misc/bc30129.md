# 程序集特性“&lt;attributename&gt;”无效: &lt;error&gt;
公共语言运行时不接受程序集中的特性。  
  
 **错误 ID:** BC30129  
  
### 更正此错误  
  
1.  从声明中删除该特性或将其替换为对上下文有效的特性。  
  
2.  检查引用的错误消息并采取适当的操作。
