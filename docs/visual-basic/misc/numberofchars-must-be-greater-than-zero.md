# NumberOfChars 必须大于 0
使用 `TextFieldParser` 对象的 `PeekChars` 方法时，必须提供一个大于 `0` 的 `NumberOfChars` 值。  
  
### 更正此错误  
  
-   将 `NumberOfChars` 更改为大于 `0` 的值。
