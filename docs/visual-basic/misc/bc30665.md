# “Throw”操作数必须从“System.Exception”派生
提供给 `Throw` 的参数必须是 `System.Exception` 的实例，或从 `System.Exception` 派生的类的实例。  
  
 **错误 ID：**BC30665  
  
### 更正此错误  
  
-   使用派生自 `System.Exception` 的参数，如下面的示例所示。  
  
    ```  
    Throw New System.Exception("This is an error.")  
    ```
