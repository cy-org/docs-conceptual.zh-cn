# 表达式“&lt;name&gt;”不是过程，但作为过程调用的目标出现
你已使过程调用成为了一个非过程的表达式。  
  
### 更正此错误  
  
1.  请确保过程的名称拼写正确。  
  
2.  为该过程提供 \<`name`\> 调用。
