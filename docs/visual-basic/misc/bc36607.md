# 应为“In”
已经指定 `From` 或 `Aggregate` 子句，但没有使用 `In` 运算符。 使用 `In` 运算符来标识要查询的集合。  
  
 **错误 ID：**BC36607  
  
### 更正此错误  
  
1.  将 `In` 运算符和键字段添加到 `From` 或 `Aggregate` 子句。 下面是一个示例：  
  
    ```vb#  
    Dim names = From pers In people Select pers.FirstName, pers.LastName  
    ```
