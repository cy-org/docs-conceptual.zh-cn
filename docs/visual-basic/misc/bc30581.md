# “AddressOf”表达式无法转换为“&lt;typename&gt;”，因为“&lt;typename&gt;”不是委托类型
语句试图将 `AddressOf` 表达式转换为不是委托类型的类型。  
  
 `AddressOf` 运算符创建引用特定过程的过程委托实例。`AddressOf` 可用作委托构造函数的操作数，或可用于可由编译器确定委托类型的上下文中。  
  
 **错误 ID：**BC30581  
  
### 更正此错误  
  
-   将目标类型更改为委托类型。
