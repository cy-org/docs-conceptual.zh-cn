# 由“DefaultEvent”特性指定的“&lt;eventname&gt;”事件不是该类的公共可访问事件。
<xref:System.ComponentModel.DefaultEventAttribute> 特性必须指定应用了该特性的类中可公开访问的事件的名称。  
  
 **错误 ID：**BC30770  
  
### 更正此错误  
  
1.  确保该类定义一个可公开访问的事件。  
  
2.  确保类中事件的名称与 <xref:System.ComponentModel.DefaultEventAttribute> 特性所指定的名称相匹配。
