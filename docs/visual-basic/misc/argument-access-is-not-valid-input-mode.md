# 参数“Acces”无效。 输入模式的有效值为“OpenAccess.Read”和“OpenAccess.Default”
访问值在 `Input` 模式中无效。  
  
### 更正此错误  
  
-   将值更改为 `OpenAccess.Read` 或 `OpenAccess.Default`。
