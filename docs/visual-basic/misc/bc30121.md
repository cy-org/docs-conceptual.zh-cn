# “Inherits”只能在“Class”语句中出现一次，并且只能指定一个类
多个 `Inherits` 语句将出现在同一个类中，或 `Inherits` 语句会指定多个类。 类可以仅从一个基类继承。  
  
 **错误 ID：**BC30121  
  
### 更正此错误  
  
-   删除任何额外 `Inherits` 语句，并确保其余的 `Inherits` 语句仅指定一个基类。
