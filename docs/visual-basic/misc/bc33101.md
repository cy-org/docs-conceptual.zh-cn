# 类型“&lt;typeName&gt;”必须是值类型或约束为“Structure”的类型参数，才能与“Nullable”或可为 null 的修饰符“?”一起使用。
只有值类型（包括结构）才可声明为可为 null。  
  
```vb#  
' Valid. Dim n? As Integer Dim m As Integer? ' Not valid. ' Dim p? As Object ' Dim q As Nullable(Of Object)  
```  
  
 **错误 ID：**BC33101  
  
### 更正此错误  
  
-   删除“?”或 `Nullable`。  
  
-   使用值数据类型。
