# 特性常量“&lt;constantname&gt;”不能作为赋值的目标
尝试给特性中声明的常量赋值。  
  
 **错误 ID：**BC31510  
  
### 更正此错误  
  
-   删除错误的赋值。  
  
-   如果使用自己开发的自定义特性，请将特性中的成员重定义为字段。
