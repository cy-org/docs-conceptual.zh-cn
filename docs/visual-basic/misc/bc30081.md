# “If”必须以匹配的“End If”结束
出现 `If` 语句而没有相应的 `End If` 语句。 必须使用 `End If` 语句结束 `If` 块。  
  
 **错误 ID：**BC30081  
  
### 更正此错误  
  
1.  如果此 `If` 块属于一组嵌套的 `If` 块，请确保每个块均已正确终止。  
  
2.  将 `End If` 语句添加到 `If` 块末尾。
