# 数组初始值设定项缺少 &lt;number&gt; 个元素
在数组的初始值设定项中没有提供足够的元素。  
  
 **错误 ID：**BC30567  
  
### 更正此错误  
  
-   提供缺少的一个或多个参数。  
  
-   使用 `ReDim` 更改数组大小。
