# “WriteOnly”属性必须提供“Set”
如果某个属性声明为 `WriteOnly`，则它必须提供用于写入其值的过程。  
  
 **错误 ID：**BC30125  
  
### 更正此错误  
  
1.  请确保 `Property` 语句和 `End Property` 语句之间包含 `Set` 过程。  
  
2.  验证 `Property` 声明中的其他过程是否正确终止。
