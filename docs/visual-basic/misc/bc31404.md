# “&lt;methodname&gt;”不能隐藏声明为“MustOverride”的方法
在派生类中声明了带有 `MustOverride` 修饰符的同名属性或方法。  
  
 **错误 ID：**BC31404  
  
### 更正此错误  
  
1.  将 `Overrides` 修饰符添加到派生类中的重写属性或方法。  
  
2.  从基类的属性或方法中删除 `MustOverride` 修饰符。
