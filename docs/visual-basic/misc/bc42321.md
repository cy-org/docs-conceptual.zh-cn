# XML 的格式不正确。 不能包含文件“&lt;commentfile&gt;”
一个 XML 注释中的 `<include>` 标记引用不存在的文件或包含格式不正确的 XML 的文件。  
  
 **错误 ID：**BC42321  
  
### 更正此错误  
  
1.  验证文件是否存在。  
  
2.  验证该文件中的 XML 是否有效。
