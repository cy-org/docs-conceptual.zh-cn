# “System.Runtime.InteropServices.DllImportAttribute”不能应用于带有非空体的 Sub、Function 或 Operator
向 `Sub`、`Function` 或不为空的 `Operator` 应用了 `DllImportAttribute` 特性。  
  
 **错误 ID：**BC31522  
  
### 更正此错误  
  
-   从 `Sub`、`Function` 或 `Operator` 删除所有代码以使用此特性。
