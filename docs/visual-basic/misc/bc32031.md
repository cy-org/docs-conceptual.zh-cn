# “End Sub”必须是一行中的第一条语句
`End Sub` 语句跟在冒号 \(:\) 语句分隔符之后。`End Sub` 必须是其源行的唯一语句。  
  
 **错误 ID：**BC32031  
  
### 更正此错误  
  
-   将多个语句分隔到不同的行。
