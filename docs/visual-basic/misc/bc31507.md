# “&lt;typename&gt;”不能用作特性，因为它具有未重写的“MustOverride”方法
具有 `MustOverride` 方法的类不能用作特性。  
  
 特性类的 `MustOverride` 成员只能在重写这些成员的派生类中使用。  
  
 **错误 ID：**BC31507  
  
### 更正此错误  
  
1.  从特性类成员中删除 `MustOverride` 修饰符。  
  
2.  在派生类中实现 `MustOverride` 成员，并将此类用作特性。
