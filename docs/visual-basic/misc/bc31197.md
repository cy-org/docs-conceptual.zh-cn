# 全角字符用作 XML 分隔符无效
已定义 XML 文本，包括作为分隔符的全角字符。 全角字符也被称为宽字符或多字节字符。  
  
 **错误 ID：**BC31197  
  
### 更正此错误  
  
-   从 XML 文本定义中删除全角字符并将其替换为有效的 ANSI 分隔符字符。 有效分隔符字符包括以下字符：`<`、`>`、`=`、`:` 和 `/`。
