# 参数“Acces”无效。 输出模式的有效值为“OpenAccess.Write”和“OpenAccess.Default”
访问值在输出模式中无效。  
  
### 更正此错误  
  
1.  将值更改为 `OpenAccess.Write` 或 `OpenAccess.Default`。
