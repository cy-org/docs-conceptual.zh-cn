# 共享的 &quot;Sub New&quot; 不能声明为“&lt;specifier&gt;”
试图在`Shared Sub New` 声明中使用说明符，而这样是无效的。  
  
 **错误 ID：**BC30480  
  
### 更正此错误  
  
-   删除说明符。
