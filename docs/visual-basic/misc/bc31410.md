# 多个基接口中声明的重载方法无效
多个继承接口隐式重载相同的方法。  
  
 **错误 ID：**BC31410  
  
### 更正此错误  
  
-   使用 `Shadows` 修饰符，而不是 `Overloads` 修饰符。
