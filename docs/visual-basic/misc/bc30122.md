# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>'停止' 语句中不能使用即时窗口
`Stop` 和 `End` 语句暂停执行，且不允许在调试上下文中使用它们。  
  
 **错误 ID：** BC30122  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” `Stop``End` 窗口中发出 **或** 语句。
