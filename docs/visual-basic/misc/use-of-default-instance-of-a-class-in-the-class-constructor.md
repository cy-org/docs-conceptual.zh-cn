# 在类构造函数中使用类的默认实例可能会导致无限递归调用
已在类的构造函数中使用类的默认实例。 这可能会导致无限递归调用（也称为无限循环）。  
  
### 更正此错误  
  
-   从类构造函数中删除默认实例。
