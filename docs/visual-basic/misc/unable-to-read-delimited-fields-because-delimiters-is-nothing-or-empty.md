# 无法读取分隔字段，因为分隔符为 Nothing 或空
无法从文件读取 `TextFieldParser`，因为 `Delimiters` 属性设置为 `Nothing` 或为空。  
  
### 更正此错误  
  
-   提供 `Delimiters` 的有效值。
