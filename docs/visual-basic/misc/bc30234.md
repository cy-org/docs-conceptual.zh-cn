# “&lt;specifier&gt;”在 WithEvents 声明中无效。
`Dim` 语句包含一个关键字（如 `ReadOnly`），此关键字与 `WithEvents` 关键字结合使用时无效。  
  
 **错误 ID：**BC30234  
  
### 更正此错误  
  
1.  从 `Dim` 语句中删除无效的关键字。
