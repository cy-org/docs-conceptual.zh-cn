# 无法读取固定宽度的字段，因为 FieldWidths 为 Nothing 或为空
`TextFieldParser` 无法读取固定宽度的字段，因为 `FieldWidths` 属性被设置为 `Nothing` 或空。  
  
### 更正此错误  
  
-   将 `FieldWidths` 设置为有效值。
