# .NET Compact Framework 的目标版本不支持后期绑定重载解决方法
.NET Compact Framework 的目标版本不支持后期绑定重载解决方法。  
  
 **错误 ID：**BC30764  
  
### 更正此错误  
  
-   将类型对象的参数强制转换为相应的类型。
