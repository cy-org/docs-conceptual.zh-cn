# “System.STAThreadAttribute”和“System.MTAThreadAttribute”不能同时应用于同一方法
`System.STAThreadAttribute` 和 `System.MTAThreadAttribute` 特性互相排斥。  
  
 **错误 ID：**BC31512  
  
### 更正此错误  
  
1.  应用 `System.MTAThreadAttribute` 或 `System.STAThreadAttribute` 两者之一，而非同时应用。
