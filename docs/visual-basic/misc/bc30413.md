# “WithEvents”变量只能类型化为具有类约束的类、接口或类型参数
声明了类型化为与 `WithEvents` 一起使用的结构的变量，这不是 `WithEvents` 修饰符的有效使用。  
  
 **错误 ID：**BC30413  
  
### 更正此错误  
  
1.  使用 `AddHandler` 来处理结构中定义的事件。
