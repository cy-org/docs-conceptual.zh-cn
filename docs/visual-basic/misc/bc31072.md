# 视为错误的警告：&lt;error&gt;
如果在打开“将警告视为错误”选项的情况下运行已编译的程序集，则会遇到警告。  
  
 **错误 ID：**BC31072  
  
### 更正此错误  
  
1.  更正此警告。  
  
2.  在“将警告视为错误”选项关闭的情况下编译程序集。
