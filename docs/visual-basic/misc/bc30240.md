# “Exit”的后面必须跟有“Sub”、“Function”、“Property”、“Do”、“For”、“While”、“Select”或“Try”
`Exit` 语句包含无效的关键字。`Exit` 必须指定要将控制权从其转移到其后的语句的块，例如 `End While`。  
  
 **错误 ID：**BC30240  
  
### 更正此错误  
  
-   在 `Exit` 后添加有效关键字，或删除 `Exit` 语句。
