# 结构无法声明没有参数的非共享“Sub New”
结构内部声明的 `Sub New` 构造函数必须接受参数或用 `Shared` 修饰符声明。  
  
 **错误 ID：**BC30629  
  
### 更正此错误  
  
-   更改 `Sub New` 构造函数以便它可以接受参数。  
  
-   应用 `Shared` 修饰符以使构造函数可共享。
