# 没有为类型“&lt;typename&gt;”定义运算符“&lt;operatorname&gt;”
尝试对某个类型（如 `String`）使用没有为其定义的一元运算符（`+`、`-` 或 `Not`）。  
  
 **错误 ID：**BC30487  
  
### 更正此错误  
  
-   将一元运算符替换为适用于该表达式的相应运算符。
