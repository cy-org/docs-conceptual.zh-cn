# “&lt;procedurename&gt;”没有任何参数，并且无法对它的返回类型进行索引
对 `Function` 或 `Sub` 过程的调用提供一个或多个参数，但是该过程不采用任何参数，且其返回类型（如果它是 `Function`）不是一个数组类型。  
  
 **错误 ID：**BC32016  
  
### 更正此错误  
  
-   从过程调用中删除一个或多个参数。
