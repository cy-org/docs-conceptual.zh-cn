# “Default”只能应用于 &lt;type&gt; 中的一个属性名
尝试在类、接口或结构中对多个属性名应用 `Default`。  
  
 **错误 ID：**BC30359  
  
### 更正此错误  
  
-   删除多余的 `Default` 声明，只保留一个。
