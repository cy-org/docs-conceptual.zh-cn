# Const 声明不能具有数组初始值设定项
尝试在 `Const` 声明中使用数组初始值设定项。  
  
 **错误 ID：**BC30445  
  
### 更正此错误  
  
-   删除初始值设定项两边的括号，将其更改为有效的表达式。
