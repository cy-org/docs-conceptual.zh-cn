# 缺少事件“&lt;eventname&gt;”的 &quot;AddHandler&quot; 定义
如果事件被声明为 `Custom`，它必须提供用于添加事件处理程序的过程。  
  
 **错误 ID：**BC31130  
  
### 更正此错误  
  
1.  在 `Custom Event` 语句和 `End Event` 语句之间包含 `AddHandler` 声明。  
  
2.  验证事件声明中的其他过程是否正确终止。
