# 初始化数组的数组时，只能指定顶级数组的界限
交错数组（数组的数组）的数组初始化设置了某个较低级数组的初始长度。 在数组声明语句中只能指定顶级数组的长度。  
  
 **错误 ID：**BC32014  
  
### 更正此错误  
  
1.  删除除顶级数组以外的其他所有数组的长度定义。  
  
2.  在使用 `New` 关键字在后续赋值语句中设置较低级别数组的初始长度。
