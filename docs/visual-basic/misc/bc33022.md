# 此转换运算符的参数类型或返回类型必须为包含类型。
转化运算符的定义使用在其中定义了此运算符的类或结构之外的类型来指定两个参数和返回类型。  
  
 在类或结构中定义转换运算符时，该运算符必须转换为\/自该类或结构的类型。  
  
 **错误 ID：**BC33022  
  
### 更正此错误  
  
-   将参数类型或返回类型更改为在其中定义了运算符的类或结构的类型。
