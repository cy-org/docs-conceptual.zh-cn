# 文件的打开模式未设置为有效值
为文件的打开模式提供的值无效。 下表显示 <xref:Microsoft.VisualBasic.OpenMode> 枚举的有效值。  
  
|值|模式|  
|-------|--------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
### 更正此错误  
  
-   验证为文件的打开模式提供的值。
