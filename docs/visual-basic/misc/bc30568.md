# 数组初始值设定项拥有的元素太多(&lt;number&gt; 个)
你的数组初始值设定项包含过多元素。  
  
 **错误 ID：**BC30568  
  
### 更正此错误  
  
-   使用 `ReDim` 更改数组大小。
