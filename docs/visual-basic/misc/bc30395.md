# “&lt;specifier&gt;”在结构声明中无效
你使用了在 `Structure` 声明上无效的说明符，例如 `Dim` 或 `Override`。  
  
 **错误 ID：**BC30395  
  
### 更正此错误  
  
-   删除说明符。
