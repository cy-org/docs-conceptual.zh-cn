# 引用的“&lt;membername&gt;”的值为 &quot;Nothing&quot;
引用的项未被分配值，或分配的值为 `Nothing`。  
  
 **错误 ID：**BC30759  
  
### 更正此错误  
  
-   验证项名称的拼写及设置是否都正确。
