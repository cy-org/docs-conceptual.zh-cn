# XML 名称开头不允许出现字符“character”(0x&lt;hexadecimal&gt;)
XML 文本或 XML 轴属性的名称开头包含了不受支持的字符。  
  
 **错误 ID：**BC31169  
  
### 更正此错误  
  
-   从 XML 文本或 XML 轴属性的名称开头删除不受支持的字符。
