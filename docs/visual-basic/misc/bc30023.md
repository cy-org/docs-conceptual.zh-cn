# 声明为“WriteOnly”的属性不能有“Get”
`Get` 过程会读取属性的值。 不能读取 `WriteOnly` 属性。  
  
 **错误 ID：**BC30023  
  
### 更正此错误  
  
-   删除 `Property` 语句中的 `WriteOnly` 关键字，或者删除属性正文中的 `Get` 过程。
