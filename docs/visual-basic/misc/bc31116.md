# “RemoveHandler”声明必须以匹配的“End RemoveHandler”结束
`RemoveHandler` 声明必须以 `End RemoveHandler` 语句结束。  
  
 **错误 ID：**BC31116  
  
### 更正此错误  
  
-   检查 `RemoveHandler` 声明是否以 `End RemoveHandler` 语句结束。
