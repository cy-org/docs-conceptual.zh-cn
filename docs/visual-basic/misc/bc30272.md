# “&lt;argumentname&gt;”不是“&lt;methoddeclaration&gt;”的参数
过程调用提供一个不是由过程定义的附加参数。  
  
 **错误 ID：**BC30272  
  
### 更正此错误  
  
-   从参数列表中删除多余的参数。
