# 未找到指定的 DLL 函数
已找到用户库引用中的动态链接库 \(DLL\)，但是未在该 DLL 中找到指定的 DLL 函数。  
  
### 更正此错误  
  
1.  在函数声明中指定有效序号。  
  
2.  请确保 DLL 的名称和别名正确。
