# 属性“&lt;propertyname&gt;”必须声明为“默认”，因为它会覆盖基“&lt;name&gt;”上的默认属性
代码中的 `Property` 语句将覆盖基本类、结构或接口中的 `Default` 属性，因此需要声明 `Default`。  
  
 **错误 ID：**BC30360  
  
### 更正此错误  
  
-   声明 `Default` 属性。
