# 当前块中已声明了局部变量“&lt;variablename&gt;”。
在同一个块（可以是过程或控制块，如 `If...Then`）中声明了具有相同名称的两个变量。  
  
 **错误 ID：**BC30288  
  
### 更正此错误  
  
-   将其中一个声明移动到其他块，或更改其中某个变量的名称。
