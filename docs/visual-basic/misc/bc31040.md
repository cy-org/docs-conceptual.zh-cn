# “&lt;membername&gt;”存在于多个基接口中
“\<membername\>”存在于多个基接口中。 请使用在“Implements”子句中声明“\<membername\>”的接口的名称，而不要使用派生接口的名称。  
  
 此接口从多个接口继承具有相同名称的成员，产生了多义性。  
  
 **错误 ID：**BC31040  
  
### 更正此错误  
  
-   使用 `Implements` 子句中的定义接口名称而不是派生接口的名称。
