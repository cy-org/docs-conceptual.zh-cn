# Option Strict On 禁止将 Object 类型的操作数用于运算符“&lt;运算符名称&gt;”
为对象变量定义的唯一运算符是 `Is` 和 `TypeOf...Is`。 当 `Option Strict` 是 `On` 时，所有操作数必须都属于为给定运算符定义的数据类型。  
  
 **错误 ID：**BC30038  
  
### 更正此错误  
  
-   使用适当的类型转换函数（如 `CInt` 或 `CStr`）将操作数转换为对运算符定义的数据类型。
