# “End Function”前面必须是匹配的“Function”
代码中的 `End Function` 语句前面没有与其匹配的 `Function` 过程定义。  
  
 **错误 ID：**BC30430  
  
### 更正此错误  
  
1.  删除多余的 `End Function` 语句。  
  
2.  提供缺少的 `Function` 过程（如果有）。  
  
3.  将 `End Function` 移到代码中的适当位置。
