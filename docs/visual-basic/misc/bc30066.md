# &quot;Exit Property&quot; 在 Function 或 Sub 中无效
`Exit Property` 出现在 `Function` 过程或 `Sub` 过程中。`Exit` 语句必须与它所在的块匹配。  
  
 **错误 ID：**BC30066  
  
### 更正此错误  
  
-   根据需要，用 `Exit Function` 或 `Exit Sub` 语句替换 `Exit Property`。
