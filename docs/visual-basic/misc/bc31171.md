# XML 注释中不允许出现字符序列“--”
XML 注释文本包含字符序列“\-\-”，这不受支持。  
  
 **错误 ID：**BC31171  
  
### 更正此错误  
  
-   从 XML 注释文本中删除字符序列“\-\-”。
