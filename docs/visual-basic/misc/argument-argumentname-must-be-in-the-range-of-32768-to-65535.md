# 参数“&lt;argumentname&gt;”必须在 -32768 到 65535 的范围内
参数的值不在 \-32,768 到 65,535 的允许范围内。  
  
### 更正此错误  
  
1.  确保参数的值在有效范围内。  
  
2.  如果参数由某个计算提供，则检查该计算所处理的值。
