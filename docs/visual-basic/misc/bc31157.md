# XML 特性“attribute1”必须在 XML 特性“attribute2”之前出现
按无效顺序指定了 XML 文档文本中的特性。 XML 文档文本的有效特性顺序基于 XML 1.0 规范。 例如，XML 文档文本的 `encoding` 特性必须紧跟在 `version` 特性的后面。  
  
 **错误 ID：**BC31157  
  
### 更正此错误  
  
-   使用 XML 1.0 规范中指定的准则更新 XML 文档文本的特性顺序。
