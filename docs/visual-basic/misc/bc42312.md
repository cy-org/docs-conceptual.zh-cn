# XML 注释块不能与任何支持应用 XML 文档注释的语言元素相关联
XML 注释块不能与任何有效的语言元素相关联。 将忽略该 XML 注释。 如果注释块中的 XML 标记没有任何匹配的语言元素，也会发生此错误。  
  
 **错误 ID:** BC42312  
  
### 更正此错误  
  
-   查找并删除多余的 XML 标记，或者提供其描述的语言元素。
