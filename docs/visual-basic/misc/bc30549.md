# 特性“&lt;attributename&gt;”不能应用于模块
尝试将特性应用于其 `AttributeUsageAttribute` 未指定 `AttributeTargets.Module` 的模块。 声明该特性时，它未定义为可应用于该模块。  
  
 **错误 ID：**BC30549  
  
### 更正此错误  
  
1.  检查特性声明，并指定 `AttributeTargets.Module` 或 `AttributeTargets.All`。
