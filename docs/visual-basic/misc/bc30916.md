# 类型“&lt;typename&gt;”直接或者间接从自身继承，因此不受支持
类或接口从自身继承，或从最终继承自该类或结构的另一个类或接口继承。  
  
 Visual Basic 不支持循环继承。  
  
 **错误 ID：**BC30916  
  
### 更正此错误  
  
-   更改继承结构，以使其基于不从其他任何类或接口继承的基类或接口构建。
