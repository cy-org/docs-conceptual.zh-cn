# 应为 &quot;As&quot;、逗号或 &quot;)&quot;
已提供的类型参数缺少 `As`、逗号 \(`,`\) 或右括号 \( `)` \)。  
  
 **错误 ID：**BC32100  
  
### 更正此错误  
  
-   提供缺少的 `As`、逗号或右括号。
