 这些示例中所使用的 `FillDataSet` 方法已在 [向数据集中加载数据](../../../../docs/framework/data/adonet/loading-data-into-a-dataset.md) 中指定。  
  
 本主题中的示例使用 AdventureWorks 示例数据库中的 Contact、Address、Product、SalesOrderHeader 和 SalesOrderDetail 表。
