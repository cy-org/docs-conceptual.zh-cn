# -（减）(Entity SQL)
两个数字相减。  
  
## 语法  
  
```  
  
expression  
-  
expression  
  
```  
  
## 参数  
 `expression`  
 任何一种数值数据类型的任何有效表达式。  
  
## 结果类型  
 对这两个参数进行隐式类型提升而产生的数据类型。 有关隐式类型提升的详细信息，请参阅 [类型系统](../../../../../../docs/framework/data/adonet/ef/language-reference/type-system-entity-sql.md)。
