## 本节内容  
 [自定义工作流设计体验](../../../docs/framework/windows-workflow-foundation//customizing-the-workflow-design-experience.md)  
 指明在 [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)] 中对用于设计自定义活动以及重新承载 [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] 的方案进行了极大地简化。现在，开发和部署变得更轻松、更灵活，因为新的活动设计器编程模型是在 [!INCLUDE[avalon1](../../../includes/avalon1-md.md)] 之上构建的。
