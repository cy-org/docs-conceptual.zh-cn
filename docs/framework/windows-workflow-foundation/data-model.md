# Windows Workflow Foundation 数据模型
Windows Workflow Foundation 数据模型包含三个概念：变量、参数和表达式。变量表示数据存储区，参数表示流入和流出活动的数据。使用可引用变量的表达式来绑定（赋值）参数。
