# 每秒排队病毒消息数
计数器名称：Queued Poison Messages Per Second（每秒排队病毒消息数）。  
  
## 描述  
 此服务中每秒由排队传输标记为病毒消息的消息数。  
  
 此计数器为性能计数器类型 [PERF\_COUNTER\_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649)（可能为英文网页），可使用以下公式计算其值：  
  
 \(N 1 \- N 0 \) \/ \( \(D 1 \-D 0 \) \/ F\)
