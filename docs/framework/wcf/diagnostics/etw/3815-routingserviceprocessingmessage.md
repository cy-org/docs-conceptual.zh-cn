# 3815 - RoutingServiceProcessingMessage
## 属性  
  
|||  
|-|-|  
|ID|3815|  
|关键字|RoutingServices|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当路由服务正在处理某一消息时，将发出此事件。  
  
## 消息  
 路由服务正在处理事务 %4 中收到的 ID 为“%1”、操作为“%2”、入站 URL 为“%3”的消息。
