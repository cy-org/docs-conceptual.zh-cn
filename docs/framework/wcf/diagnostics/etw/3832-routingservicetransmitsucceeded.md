# 3832 - RoutingServiceTransmitSucceeded
## 属性  
  
|||  
|-|-|  
|ID|3832|  
|关键字|RoutingServices|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当路由服务使用指定 ID 成功传送消息时发出此事件。  
  
## 消息  
 路由服务已成功将 ID 为“%1”的消息 \[操作 %2\] 传输到“%3”。
