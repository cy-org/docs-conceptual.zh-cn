# 4807 - DiscoveryMessageWithInvalidReplyTo
## 属性  
  
|||  
|-|-|  
|ID|4807|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当发现消息由于具有无效 ReplyTo 地址而被丢弃时发出此事件。  
  
## 消息  
 messageId 为“%1”的发现请求消息由于具有无效 ReplyTo 地址而被丢弃。
