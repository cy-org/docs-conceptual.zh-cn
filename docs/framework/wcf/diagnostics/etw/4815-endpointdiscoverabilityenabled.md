# 4815 - EndpointDiscoverabilityEnabled
## 属性  
  
|||  
|-|-|  
|ID|4815|  
|关键字|发现|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当启用了终结点的可发现性时，将发出此事件。  
  
## 消息  
 已启用 EndpointAddress 为“%1”和 ListenUri 为“%2”的终结点的可发现性。
