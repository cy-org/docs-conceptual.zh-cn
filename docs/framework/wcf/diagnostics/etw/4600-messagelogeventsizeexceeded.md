# 4600 - MessageLogEventSizeExceeded
## 属性  
  
|||  
|-|-|  
|ID|4600|  
|关键字|WCFMessageLogging|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当因为消息超出 ETW 事件大小而无法记录时发出此事件。  
  
## 消息  
 无法记录消息，因为它超出了 ETW 事件大小
