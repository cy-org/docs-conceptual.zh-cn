# 226 - IdleServicesClosed
## 属性  
  
|||  
|-|-|

|关键字|HealthMonitoring、WebHost|  
|级别|LogAlways|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/分析|
