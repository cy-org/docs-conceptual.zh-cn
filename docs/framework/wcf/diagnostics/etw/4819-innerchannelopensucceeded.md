# 4819 - InnerChannelOpenSucceeded
## 属性  
  
|||  
|-|-|  
|ID|4819|  
|关键字|发现|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当 DiscoveryClientChannel 成功地发现终结点并使用它打开通道时，发出此事件。  
  
## 消息  
 DiscoveryClientChannel 成功发现了一个终结点并使用该终结点打开了通道。  使用 EndpointAddress\=“%1”和 Via\=“%2”将客户端连接到服务。
