# 1442 - ReceiveRetryCountReached
## 属性  
  
|||  
|-|-|  
|ID|1442|  
|关键字|配额|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/分析|  
  
## 描述  
 当对于 MSMQ 消息达到接收重试计数时，发出此事件。  
  
## 消息  
 对于 id 为“%1”的 MSMQ 消息，已达到接收重试计数限制
