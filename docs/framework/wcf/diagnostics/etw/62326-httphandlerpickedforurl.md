# 62326 - HttpHandlerPickedForUrl
## 属性  
  
|||  
|-|-|  
|ID|62326|  
|关键字|WebHost|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 在选取 HTTP 处理程序为对此 URL 做出的所有请求提供服务时，发出此事件。  
  
## 消息  
 URL“%1”承载含有根元素类型“%2”的 XAML 文档。  选取 HTTP 处理程序类型“%3”为对此 URL 做出的所有请求提供服务。
