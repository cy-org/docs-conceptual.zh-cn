# 3816 - RoutingServiceTransmittingMessage

|||

## 描述  
 当路由服务正在传输某一消息时，发出此事件。  
  
## 消息  
 路由服务正在将 ID 为“%1”的消息 \[操作 %2\] 传输到“%3”。
