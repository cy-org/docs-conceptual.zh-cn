# 3413 - HttpPipelineBeginProcessResponseStart
## 属性  
  
|||  
|-|-|

|关键字|HTTP|  
|级别|详细|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当 Http 消息处理程序开始以异步方式处理响应时，发出此事件。  
  
## 消息  
 Http 消息处理程序已开始以异步方式处理响应。
