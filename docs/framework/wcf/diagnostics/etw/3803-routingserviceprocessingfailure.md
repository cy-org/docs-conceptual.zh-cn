# 3803 - RoutingServiceProcessingFailure
## 属性  
  
|||  
|-|-|  
|ID|3803|  
|关键字|RoutingServices|  
|级别|错误|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 在终结点上处理消息时，路由服务失败，此时发出此事件。  
  
## 消息  
 在地址为“%1”的终结点上处理消息时，路由服务失败。
