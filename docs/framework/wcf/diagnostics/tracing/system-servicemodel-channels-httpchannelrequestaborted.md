# System.ServiceModel.Channels.HttpChannelRequestAborted
未能通过 HTTP 发送请求消息。  
  
## 描述  
 此警告级别跟踪指示已中止 HTTP 通道请求。  此跟踪中的其他信息将显示原因或传递到 Abort 方法的消息。
