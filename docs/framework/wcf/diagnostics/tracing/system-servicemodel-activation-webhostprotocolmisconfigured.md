# System.ServiceModel.Activation.WebHostProtocolMisconfigured
协议在 Windows 进程激活服务 \(WAS\) 中配置不正确。  
  
## 描述  
 此错误在传输协议配置不正确时发生。  
  
## 疑难解答  
 检查并修正 web.config 中的传输配置参数。
