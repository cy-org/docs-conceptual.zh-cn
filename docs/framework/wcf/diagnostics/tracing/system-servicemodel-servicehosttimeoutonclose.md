# System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## 说明  
 ServiceHost 的关闭操作已超时。这表示某些会话通道没有被客户端正确关闭，可能已中止。请修改默认时间以调整关闭超时。
