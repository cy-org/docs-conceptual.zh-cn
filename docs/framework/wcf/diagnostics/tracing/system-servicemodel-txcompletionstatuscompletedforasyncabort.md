# System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
指定操作的指定事务由于异步中止而完成。  
  
## 说明  
 由于另一名参与者赞成中止，发生超时，或事务的参与者内部出现另一错误，因此当前事务被中止。  
  
## 疑难解答  
 如果这是意外中止，请检查所有系统日志，以确定发生该中止的真正原因。
