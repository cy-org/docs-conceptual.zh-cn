# Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
协议服务从无法识别的可变参与者接收到已准备或重播消息。已向参与者返回错误，声明事务的结果不确定。  
  
## 说明  
 在本地事务管理器从它已经忘记的可变参与者接收到已准备或重播消息时进行跟踪。  
  
## 疑难解答  
 调查来自可变参与者的最新消息的可能原因。
