# Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
无法创建事务。  
  
## 描述  
 当 MSDTC 无法创建事务时，将生成此跟踪。  这可能是由资源不足、日志空间不够或其他错误引起的。  
  
## 疑难解答  
 检查跟踪消息中的状态字符串以确定是否存在任何可操作的项。
