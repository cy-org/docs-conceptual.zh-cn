# System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
在指定的时间限制内，安全握手并未成功。  
  
## 描述  
 此跟踪在尝试建立安全邻居连接时发生。  
  
## 疑难解答  
 确保所有邻居都具有适合的安全凭据并在 1 分钟之内对所有消息做出响应。
