# Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
WS\-AT 协议服务无法向其协调程序发送注册消息  
  
## 说明  
 在本地 TransactionManager 由于不能发送消息而无法向其上级 TransactionManager 注册时跟踪。  
  
## 疑难解答  
 检查跟踪消息以查找导致消息发送失败的异常
