# System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## 描述  
 已到达一个 MaxConcurrentCalls\/Instances\/Sessions 控制器且队列中有一个消息。  
  
 您可以根据预期的和实际的生产负载使用此跟踪调整控制器值。
