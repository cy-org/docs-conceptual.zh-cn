# System.ServiceModel.Channels.SocketConnectionCreate
System.ServiceModel.Channels.SocketConnectionCreate  
  
## 说明  
 此跟踪在客户端执行的第一个“处理操作”活动中以及服务上的“接收字节”活动中发出。此跟踪提供本地和远程 IP 地址，以信息级别发出。
