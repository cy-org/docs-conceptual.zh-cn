# Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata
在协调器程序恢复日志项中发现包含无效或不兼容的元数据的终结点引用。  
  
## 说明  
 系统无法为已恢复的协调程序登记创建代理。
