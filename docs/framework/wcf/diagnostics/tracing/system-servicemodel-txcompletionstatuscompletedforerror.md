# System.ServiceModel.TxCompletionStatusCompletedForError
由于未处理的执行异常，导致指定操作的指定事务处于完成状态。  
  
## 说明  
 在尝试完成当前事务过程中发生错误时跟踪。在将答复或错误发送给调用方之前，会发生此情况。  
  
## 疑难解答  
 检查跟踪的消息，以查找异常消息和所有可操作的项。
