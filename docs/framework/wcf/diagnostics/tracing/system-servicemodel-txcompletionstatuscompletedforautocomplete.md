# System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
由于 `OperationBehaviorAttribute.TransactionAutoComplete` 成员被设置为 `true`，因此指定操作的指定事务已被完成。  
  
## 说明  
 当基础结构自动代表用户完成事务时进行跟踪。
