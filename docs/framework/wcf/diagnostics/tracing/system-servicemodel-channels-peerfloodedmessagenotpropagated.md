# System.ServiceModel.Channels.PeerFloodedMessageNotPropagated
消息已由 PeerNode 接收，但尚未传播到邻近节点。  
  
## 说明  
 在处理传入溢满消息期间出现此跟踪。  
  
## 疑难解答  
 检查消息传播筛选器是否存在。
