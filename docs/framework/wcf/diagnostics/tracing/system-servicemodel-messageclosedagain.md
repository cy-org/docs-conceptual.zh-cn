# System.ServiceModel.MessageClosedAgain
System.ServiceModel.MessageClosedAgain  
  
## 说明  
 已再次关闭消息。  
  
 消息只应关闭一次。如果用户扩展代码发出此跟踪，则指示用户扩展代码正在关闭一个已经关闭的消息。如果通过产品代码发出此跟踪，则指示用户扩展代码可能正在过早地关闭消息。
