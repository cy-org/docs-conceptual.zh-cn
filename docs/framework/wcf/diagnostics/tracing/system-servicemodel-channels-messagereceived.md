# System.ServiceModel.Channels.MessageReceived
System.ServiceModel.Channels.MessageReceived  
  
## 说明  
 通过通道接收的消息。  
  
 如果在信息级别或详细级别启用了 `System.ServiceModel` 跟踪源，则将在接收到 [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] 消息时发出此跟踪。此跟踪对于在活动图形视图中查看消息相关箭头而言是必需的。
