# System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## 说明  
 请求\/答复操作没有任何答复消息。  
  
 这是一个双向操作；但是，没有要发送到客户端的答复。在开发过程中，可以使用此跟踪进行调试。
