# System.ServiceModel.Channels.HttpsClientCertificateInvalid
客户端证书无效。  
  
## 说明  
 此跟踪指示 HTTPS 侦听器已发现客户端提供的证书无效。HTTPS 侦听器尝试使用此证书验证客户端的真实性。如果承载服务的服务器未能识别证书颁发机构，则证书可能无效。
