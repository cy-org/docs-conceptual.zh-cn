# Microsoft.Transactions.TransactionBridge.CoordinatorStateMachineFinished
用于协调器登记的状态机已进入已完成状态。  
  
## 说明  
 本地事务管理器认为高级协调器登记已完成 2pc 处理时跟踪。登记的结果可以是“已提交”、“已中止”或“已忘记”。本地事务管理器在“准备”过程中对“只读”投票时也会进行跟踪。
