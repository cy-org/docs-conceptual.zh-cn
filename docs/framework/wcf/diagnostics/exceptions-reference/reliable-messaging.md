# 可靠消息传递
本主题列出由“可靠消息”生成的所有异常。  
  
## 异常列表  
  
|资源代码|资源字符串|  
|----------|-----------|  
|InvalidBufferRemaining|消息被具有指定剩余大小的缓冲区接收。剩余缓冲区的大小不能小于零。|
