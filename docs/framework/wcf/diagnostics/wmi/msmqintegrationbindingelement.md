# MsmqIntegrationBindingElement
MsmqIntegrationBindingElement  
  
## 语法  
  
```  
class MsmqIntegrationBindingElement : MsmqBindingElementBase  
{  
  string SerializationFormat;  
};  
```  
  
## 方法  
 MsmqIntegrationBindingElement 类未定义任何方法。  
  
## 属性  
 MsmqIntegrationBindingElement 类具有以下属性：  
  
### SerializationFormat  
 数据类型：String  
  
 访问类型：只读  
  
 绑定用于序列化消息的格式。
