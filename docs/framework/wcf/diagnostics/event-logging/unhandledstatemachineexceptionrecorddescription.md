# UnhandledStateMachineExceptionRecordDescription
ID：136  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 描述  
 此事件指示状态机处理事件时引发了一个未处理的异常。  该事件列出了事务 ID、状态机名称、当前状态、历史记录、登记 ID、异常、进程名称和进程 ID。
