# FailedToCreateMessageLoggingTraceSource
Id：115  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## 说明  
 此事件指示消息日志记录无法创建跟踪源。此事件列出异常、进程名称和进程 ID。
