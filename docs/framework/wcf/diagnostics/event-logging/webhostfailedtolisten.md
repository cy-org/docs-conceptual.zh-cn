# WebHostFailedToListen
Id：112  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## 说明  
 此事件指示在尝试侦听特定 URL 时出现错误。已终止该辅助进程。此事件将列出异常、进程名称和进程 ID。
