# TransactionBridgeRecoveryFailure
Id：146  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 说明  
 此事件指示恢复期间 MSDTC TransactionBridge 失败。这是错误状态，因此 MSDTC 服务将终止。此事件列出了异常、进程名称和进程 ID。
