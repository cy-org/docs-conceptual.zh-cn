# CoordinatorRecoveryLogEntryCorrupt
Id: 139  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 说明  
 此事件指示协调程序恢复日志条目已损坏，无法进行反序列化。此错误可能导致数据丢失。此事件列出了事务 ID、恢复数据（Base64 编码）、异常、进程名称和进程 ID。
