# 组合框
<xref:System.Windows.Controls.ComboBox> 控件呈现给用户一个选项列表。  随控件展开或折叠显示和隐藏列表。  该列表在其默认状态中是折叠的，仅显示一个选项。  用户可单击按钮来查看选项的完整列表。
