# 样式和模板
Windows Presentation Foundation \(WPF\) 样式设置和模板化是指一套功能（样式、模板、触发器和演示图板），应用程序、文档或用户界面 \(UI\) 设计人员使用这些功能可以创建视觉效果更佳的应用程序，也可以对其产品的特定外观进行标准化。
