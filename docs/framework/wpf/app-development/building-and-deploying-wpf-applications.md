# 生成和部署 WPF 应用程序
生成和部署模型提供在本地和远程生成和部署应用程序的功能，包括：  
  
-   MSBuild：位于 Microsoft.Build.Tasks.Windows 命名空间中的 .NET 生成系统。  
  
-   资源：与 UI 资源一起使用。  
  
-   ClickOnce 部署：.NET 发布和部署系统。
