# WPF 应用程序中的窗口
用户通过窗口与应用程序进行交互。  窗口的基本目的是承载和显示内容。  窗口承载的内容类型取决于应用程序操作的数据的类型，其中包括媒体、可扩展应用程序标记语言 \(XAML\) 页、网页、文档、数据库表和记录以及系统信息。
