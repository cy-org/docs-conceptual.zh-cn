# WPF 中的 XAML
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] 是声明性应用程序编程标记语言。  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 实现 XAML 处理器实现，并提供 XAML 语言支持。  [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 类型得以实现，这样，它们将能够为 XAML 表示形式提供必需的类型支持。通常，可以在 XAML 标记中创建大部分 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 应用程序 UI。
