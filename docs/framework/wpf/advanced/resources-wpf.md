# 资源 (WPF)
资源是可以替换应用程序的不同位置重用的对象。  WPF 支持资源具有不同的类型。  这些资源是资源的主要两种类型:XAML 资源和资源数据文件。  XAML 资源的示例包括画笔和样式。  资源数据文件是应用程序所需的不可执行的数据文件。
