# HScrollBar 和 VScrollBar 控件（Windows 窗体）
Windows 窗体滚动条控件用于在项的长列表或大量信息中轻松导航，方法是在应用程序或控件中水平或垂直滚动。  滚动条是 Windows 界面的一种常见元素。
