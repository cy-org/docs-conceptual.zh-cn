# TextBox 控件（Windows 窗体）
Windows 窗体文本框用于获取用户输入或显示文本。  `TextBox` 控件通常用于可编辑文本，不过也可使其成为只读控件。  文本框可以显示多个行，对文本换行使其符合控件的大小以及添加基本的格式设置。  `TextBox` 控件仅允许在其中显示或输入的文本采用一种格式。
