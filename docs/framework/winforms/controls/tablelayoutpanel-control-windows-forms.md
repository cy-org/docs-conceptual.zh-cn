# TableLayoutPanel 控件（Windows 窗体）
<xref:System.Windows.Forms.TableLayoutPanel> 控件将其内容排列在网格中。  由于布局是同时在设计时和运行时执行的，因此它可随应用程序环境的变化而动态地变化。  这使得面板中的控件能够按比例调整大小，以便能够响应更改（例如父控件的大小调整或本地化产生的文本长度更改）。
