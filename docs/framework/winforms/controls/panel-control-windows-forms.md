# Panel 控件（Windows 窗体）
Windows 窗体 `Panel` 控件用于为其他控件提供可识别的分组。  通常，使用面板按功能细分窗体。  `Panel` 控件类似于 <xref:System.Windows.Forms.GroupBox> 控件；但只有 `Panel` 控件可以有滚动条，只有 <xref:System.Windows.Forms.GroupBox> 控件可显示标题。
