# ComboBox 控件（Windows 窗体）
Windows 窗体 `ComboBox` 控件用于在下拉组合框中显示数据。  默认情况下，`ComboBox` 控件分两个部分显示：顶部是一个允许用户键入列表项的文本框。  第二部分是一个列表框，它显示一个项列表，用户可从中选择一项。
