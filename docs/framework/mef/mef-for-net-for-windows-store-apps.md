# 适用于 Windows 应用商店应用的 .NET 的 MEF 命名空间
<xref:System.Composition?displayProperty=fullName> 和其子命名空间包含使用托管扩展性框架 \(MEF\) 开发可扩展的 [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] 应用程序类型。  这些命名空间是 [!INCLUDE[win8](../../../includes/win8-md.md)] 操作系统的 [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] 子集的一部分。  
  
 这些命名空间不是与 .NET Framework 一起分配的核心类库的一部分。。  若要安装这些命名空间，请在 [!INCLUDE[vs_dev11_long](../../../includes/vs-dev11-long-md.md)] 中打开项目，从**“项目”**菜单中选择**“管理 NuGet 包”**，并联机搜索 Microsoft.Composition 包。  
  
-   <xref:System.Composition?displayProperty=fullName> 提供组成 [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] 应用程序核心 MEF 的类。  
  
-   <xref:System.Composition.Convention?displayProperty=fullName> 提供支持使用带基于约定的配置模型的 MEF 的类型。  
  
-   <xref:System.Composition.Hosting?displayProperty=fullName>提供对承载应用程序开发人员有用的 MEF 类。  
  
-   <xref:System.Composition.Hosting.Core?displayProperty=fullName> 提供组合引擎内部使用的 MEF 类型。  
  
 有关 [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] 的更多信息及它包含命名空间和类型的列表，如 Windows 开发人员中心参见 [存储应用 .NET 的窗口概述](http://go.microsoft.com/fwlink/p/?LinkID=238312)。
