# &lt;privacyNoticeAt&gt;
表示一个配置元素，该元素指定 `wsFederationHttp` 绑定中使用的隐私声明。  
  
## 语法  
  
```  
  
<privacyNotice url="String"  
        version="Integer" />  
```  
  
## 类型  
 `Type`  
  
## 特性和元素  
 下列各节描述了特性、子元素和父元素。  
  
### 特性  
  
|特性|描述|  
|--------|--------|  
|`url`|一个字符串，指定隐私声明位于的 URI。|  
|`version`|一个整数，指定此隐私声明的版本。|  
  
### 子元素  
 无。  
  
### 父元素  
  
|元素|描述|  
|--------|--------|  
|[\<绑定\>](../../../../../docs/framework/misc/binding.md)|定义自定义绑定的所有绑定功能。|
